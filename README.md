# vue-automation

## 这是什么

一款开箱即用的 Vue 项目模版，基于 Vue CLI

## 特点

- 默认集成 vue-router 和 vuex
- 全局 SASS 资源自动引入
- 精灵图自动生成
- 全局组件自动注册
- CDN支持，优化打包体积
- 轻松实现团队代码规范
